## 🔍 О проекте

**ГЕРОН-АВТО** — это современный веб-сайт для автосервиса, специализирующегося на ремонте грузовых автомобилей, прицепов и диагностике транспортных средств. Проект включает в себя публичную часть сайта, личный кабинет пользователя и полноценную административную панель.

Сайт предоставляет клиентам возможность:
- Ознакомиться с перечнем услуг
- Просмотреть информацию о сотрудниках компании
- Оставить и прочитать отзывы
- Подать заявку на ремонт онлайн
- Откликнуться на вакансии компании

## 🛠 Технологии

### Backend
- **PHP 8+** — серверная логика и REST API
- **MySQL 8** — реляционная база данных
- **PDO** — безопасная работа с БД через подготовленные запросы
- **PHPMailer** — отправка email-уведомлений и кодов подтверждения
- **PHP Sessions** — управление авторизацией пользователей

### Frontend
- **HTML5 / CSS3** — разметка и стилизация
- **Vanilla JavaScript (ES6+)** — интерактивность и работа с API
- **Fetch API** — асинхронные запросы к серверу

### Инфраструктура
- **Apache / Nginx** — веб-сервер
- **SMTP (Яндекс)** — почтовый сервер для уведомлений

## 🗄 База данных

Проект использует **MySQL** базу данных `geron_auto` с кодировкой `utf8mb4`. Схема включает 5 таблиц:

```sql
geron_auto
├── users              -- Зарегистрированные пользователи
├── requests           -- Заявки на ремонт (статусы: new / in_progress / completed / cancelled)
├── reviews            -- Отзывы пользователей (рейтинг 1–5)
├── email_verification -- Коды подтверждения email при регистрации
└── resumes            -- Резюме соискателей (статусы: new / viewed / accepted / rejected)
```

### Описание таблиц

<details>
<summary><b>👤 users</b> — пользователи системы</summary>

```sql
id, name, email (UNIQUE), phone, password (hash), avatar, created_at, updated_at
```
</details>

<details>
<summary><b>🔧 requests</b> — заявки на ремонт</summary>

```sql
id, user_id (FK), service_type, vehicle_model, problem_description,
photo, preferred_date, phone, status (ENUM), admin_comment, created_at
```
</details>

<details>
<summary><b>⭐ reviews</b> — отзывы</summary>

```sql
id, user_id (FK), rating (1-5), comment, created_at
```
</details>

<details>
<summary><b>✉️ email_verification</b> — коды подтверждения</summary>

```sql
id, email, code (6 символов), expires_at, is_used, created_at
```
</details>

<details>
<summary><b>📄 resumes</b> — резюме соискателей</summary>

```sql
id, position, full_name, phone, email, call_time, age,
experience, comment, status (ENUM), created_at
```
</details>

## 🔌 API Эндпоинты

Все запросы возвращают JSON. Формат ответа:

```json
{ "success": true,  "message": "Успешно", "data": { ... } }
{ "success": false, "message": "Текст ошибки" }
```

| Метод | URL | Описание |
|-------|-----|----------|
| `POST` | `/api/send-code.php` | Отправить код подтверждения на email |
| `POST` | `/api/register.php` | Регистрация нового пользователя |
| `POST` | `/api/login.php` | Вход в систему |
| `POST` | `/api/logout.php` | Выход из системы |
| `GET` | `/api/auth.php` | Проверка текущей сессии |
| `GET/PUT` | `/api/profile.php` | Получение / обновление профиля |
| `GET/POST` | `/api/requests.php` | Список заявок / создание заявки |
| `GET/POST` | `/api/reviews.php` | Список отзывов / публикация отзыва |
| `GET` | `/api/employees.php` | Список сотрудников |
| `POST` | `/api/resume.php` | Отправка резюме на вакансию |
| `GET` | `/api/check-admin.php` | Проверка прав администратора |

## 🚀 Установка

### Требования

- PHP **8.0+**
- MySQL **8.0+**
- Веб-сервер **Apache** или **Nginx** с поддержкой PHP
- Composer (опционально, для PHPMailer)

### Шаги установки

**1. Клонирование репозитория**

```bash
git clone https://github.com/ZINEX53/Geron.git
cd Geron
```

**2. Создание базы данных**

```bash
mysql -u root -p < database.sql
```

Или выполните SQL из файла `database.sql` через phpMyAdmin / другой клиент.

**3. Настройка конфигурации**

Откройте файл `config.php` и укажите параметры подключения к БД и почте:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'geron_auto');
define('DB_USER', 'your_db_user');
define('DB_PASS', 'your_db_password');
```

**4. Установка PHPMailer**

Поместите файлы PHPMailer в директорию `vendor/PHPMailer/`:
- `PHPMailer.php`
- `SMTP.php`
- `Exception.php`

**5. Настройка веб-сервера**

Укажите корень сайта (`DocumentRoot`) на папку проекта. Убедитесь, что PHP включён и работает.

**6. Готово!**

Откройте `GERON.HTML` в браузере или настройте виртуальный хост.

## ⚙ Настройка

### Конфигурация базы данных (`config.php`)

```php
define('DB_HOST', 'localhost');   // Хост базы данных
define('DB_NAME', 'geron_auto'); // Имя базы данных
define('DB_USER', 'root');        // Пользователь БД
define('DB_PASS', '');            // Пароль БД
define('DB_CHARSET', 'utf8mb4'); // Кодировка
```

### Конфигурация почты (`config.php`)

```php
define('MAIL_HOST', 'smtp.yandex.ru'); // SMTP-сервер
define('MAIL_PORT', 587);              // Порт (STARTTLS)
define('MAIL_USER', 'your@email.ru'); // Email отправителя
define('MAIL_PASS', 'your_password'); // Пароль или app-пароль
define('MAIL_TO',   'admin@email.ru'); // Email получателя уведомлений
```

> ⚠️ **Важно:** Не публикуйте реальные данные для подключения в публичных репозиториях. Используйте `.env` файлы или переменные окружения.
